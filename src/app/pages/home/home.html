Lập trình viên có kinh nghiệm xây dựng giải pháp đa nền tảng với ReactJS và React Native. Tập trung vào việc phát triển ứng dụng ổn định, chất lượng và mong muốn đồng hành cùng đội ngũ kỹ thuật chuyên nghiệp.

Với kinh nghiệm làm việc tại Vietapp (Software Developer) và FDS (Fullstack Intern), tôi đã có cơ hội phát triển kỹ năng lập trình và hiểu rõ quy trình phát triển phần mềm chuyên nghiệp.

Những điểm mạnh của tôi:

  • ✨ Xây dựng bộ component UI riêng với React Native
  • ✨ Build & release ứng dụng mobile lên App Store và Google Play
  • ✨ Phát triển & tối ưu hóa các hệ thống CRM, FNB, Sản xuất
  • ✨ Quản lý state phức tạp với Redux Toolkit/Zustand
  • ✨ Tích hợp API với TanStack Query/Redux Thunk
  • ✨ Viết code sạch, dễ bảo trì, có performance tốt
